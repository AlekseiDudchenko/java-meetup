# Software Developers Meetup · Heidelberg / Mannheim / Karlsruhe

Community website for regular Software Developers meetups in the Rhine-Neckar region.

**Live site:** https://alekseidudchenko.github.io/java-meetup/

## About
- Regular evening meetups (every ~2 months)
- Format: 2 main talks (30 min + Q&A) + lightning talks (5 min) + networking
- Topics: JVM internals, Spring ecosystem, concurrency, architecture, real-world engineering
- Always-open CFP — talks of 5, 15, or 30 minutes welcome
- Free to attend, community-driven, no marketing

## Stack
Static HTML/CSS/JS — no build step, no dependencies, deploys directly via GitHub Pages.

## Contributing
- To suggest a talk: open an Issue with label `cfp`
- To report a bug: open an Issue with label `bug`
- To update event history: edit the events section in `index.html` and open a PR
